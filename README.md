# 🖱️ Air Mouse using ESP32

## 📌 Project Overview

This project implements an **Air Mouse system** using the ESP32 microcontroller and MPU6050 accelerometer/gyroscope sensor to control cursor movement through hand gestures.

The MPU6050 detects the motion and orientation of the hand, while the ESP32 processes this data and converts it into cursor movement, enabling a wireless and intuitive human-computer interaction system.

---

## ⚙️ Features

* Gesture-based cursor control
* Real-time motion tracking
* Wireless operation using ESP32
* Click functionality using hardware buttons
* Portable design with rechargeable battery

---

## 🧠 Working Principle

* The **MPU6050 sensor** captures hand motion (tilt, acceleration, rotation).
* The **ESP32** processes sensor data and maps it to cursor movement.
* **Buttons** are used to perform left and right click operations.
* A **battery with charging module** powers the system, making it portable.

---

## 🔩 Components Used

* ESP32
* MPU6050 (Accelerometer + Gyroscope)
* Push Buttons
* Rechargeable Battery
* Charging Module (e.g., TP4056)
* Connecting Wires
